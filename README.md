# studentPerformance_analysis

## 📌 Project Overview
This project analyzes the academic performance of 20 students using Python.  
The goal is to practice basic data analysis techniques such as data loading, feature creation, aggregation, and visualization.

---

## 📂 Dataset
The dataset contains marks of 20 students in:
- Maths
- Science
- English

Two new features were created:
- **Total** – Sum of all subject marks
- **Average** – Mean score of each student
- **Grade** – Assigned based on average marks

---

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 🔍 Key Analysis Performed
- Identified the topper using maximum total marks
- Calculated subject-wise averages
- Determined the best performing subject
- Assigned grades based on performance
- Visualized:
  - Subject-wise average marks
  - Top 5 students
  - Grade distribution

---
